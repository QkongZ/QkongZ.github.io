Debian中的一些常用命令
2023-07-24

系统软件相关

更新系统所有
apt update
更新现有软件
apt upgrade -y
更新软件依赖关系更新现有软件删除依赖以外的软件
apt full-upgrade -y
安装或更新指定软件如：curl wget
apt install -y curl wget
删除指定软件如：curl wget
apt remove -y curl wget
删除debian用户
sudo deluser debian
